# MKG Freight Lines - Local Dev Server

A local Express development server has been set up to serve the static landing page from the `SDMLS-main` directory.

## How to Run Locally

1. **Install Dependencies (if not already done):**
   ```bash
   npm install
   ```

2. **Start the Local Server:**
   ```bash
   npm run dev
   ```

   The website will be available at:
   - **http://localhost:3000** (or next available port if 3000 is occupied).
