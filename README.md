# MKG Freight Lines - Local Dev Server

This project is configured with a local Express development server to serve the static landing page from the root of the repository.

## How to Run Locally

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start the Local Server:**
   ```bash
   npm run dev
   ```

   The website will be available at:
   - **http://localhost:3000** (or next available port if 3000 is occupied).

## Repository Structure

- `index.html`: The main homepage file.
- `career.html`: The Careers page file.
- `server.js`: The local Express server script.
- `package.json`: NPM scripts and dependencies.
- `*.jpg` / `*.png` / `*.ico`: Brand assets and images.
